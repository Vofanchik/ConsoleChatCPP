# ConsoleChatCPP (Консольный чат)
---
## Описание
 Программа реализует простой консольный чат. Пользователь может зарегистрироваться, введя для этого пароль и логин. Войдя в чат отправлять сообщения кому-либо из списка зарегистрированных пользователей или всем зарегистрированным пользователям.

### Реализация
Созданы три класса ```User()``` ```Message()``` и ```Chat()```.
Класс ```User()``` позволяет создавать объекты, которые содержат персональные данные пользователя (логин, пароль).  Класс ```Message()``` отвечает за отправку и прием сообщений.  Класс ```Chat()``` включает в себя список пользователей, списки сообщения и текущего активного пользователя. 
Организована система меню, которая позволяет выбрать режим работы чата (регистрация пользователя, вход в чат, отправить сообщение, прочитать сообщение). После запуска меню, с возможными вариантами продолжения работы.
